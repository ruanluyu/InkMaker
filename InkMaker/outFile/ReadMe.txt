===============================================（简体中文）
InkMaker v1.0

在屏幕上按住鼠标并 缓慢 拖动。
按下键盘“S”键来使渲染停止；
按下键盘“C”键清空屏幕；
按下键盘“R”键保存渲染的序列帧（会固定保存在当前目录下的output文件夹）。

渲染器崩溃时需要按“S”来重置渲染器。


InkMaker v2.0

上下键来切换要更改的变量，左右键更改当前变量的数值，回车键来重置变量。
变量Dying Probability （默认0.4，范围0-1）像素死亡几率，可以控制扩散范围。
变量Fading Probability （默认0.3，范围0-1）像素褪色几率，可以控制颜色褪色速度，数值越高褪色越快。
变量Fading Ratio（默认0.9，范围0-1）像素褪色百分比，数值=褪色后色值/褪色前色值。数值越高颜色分离度越低，数值越低颜色层次越明显。
变量Min Level （默认25，范围0-255）最低像素色值，当像素低于这个值的时候死亡，是控制扩散结束判定标准的变量。数值越高扩散结束的越快。

===============================================（English）
InkMaker v1.0

Press left button on screen and drag in low speed.
Press 's' to stop render;
Press 'c' to clear screen;
Press 'r' to save frames(will be saved in a folder named "output" in the same folder with this ReadMe text file).

When render breaked down,press 's' to reset render.

InkMaker v2.0

Press up or down to change the property which you want to change,and press left or right to change the value of it,press Enter to reset to default value.
Property "Dying Probability"(default 0.4,from 0 to 1),probability of death of a pixel ,can also control the range of ink spreading.
Property "Fading Probability"(default 0.3,from 0 to 1),probability of fading.When it is high, it fades in a fast way.
Property "Fading Ratio"(default 0.9,from 0 to 1), ratio of how much it is going to fade.When it is high, it will cut just a bit of brightness level down in every fading process.
Property "Min Level"(default 25 ,from 0 to 255), the minimum of brightness.When brightness of a pixel is lower than this value,it will die.
